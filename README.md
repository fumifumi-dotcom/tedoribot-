# 🌐 Money-Tools: Webプロジェクト統合管理ディレクトリ

このフォルダ（`money-tools`）は、今後展開される様々なWebサイト、Bot、アプリなどの全プロジェクトを統合管理するハブ（親ディレクトリ）です。

ファイルが迷子になるのを防ぐため、以下の階層ルールに従って管理されます。

## 📂 ディレクトリ階層ルール

新しいプロジェクト（サイトやアプリ）を作る際は、必ず直下に「プロジェクトごとの専用フォルダ」を作成し、関連ファイルはすべてその中に閉じ込めます。

```text
money-tools/
│
├── README.md               <-- 📍 今見ているファイル（全プロジェクトの目次）
│
├── tedori_dotcom/          <-- 稼働中: 手取り計算.com（Webサイト ＋ X自動投稿Bot）
│   ├── index.html          # サイト本体
│   ├── auto_x_poster.py    # X自動投稿システム
│   ├── sns_content/        # 画像や生成されたデータ類
│   └── ...
│
├── (将来のプロジェクトA)/  <-- 予定: 新しいサービス
│   ├── index.html
│   └── ...
│
└── (将来のプロジェクトB)/
```

## 🚀 現在稼働中のプロジェクト一覧

### 1. 手取り計算.com (`tedori_dotcom/`)
- **概要**: 額面から手取りを計算するシミュレーションサイト。SNS(X)集客用の完全自動Botシステムを内包。
- **自動化設定**: Mac標準の `launchd` を使用し、毎日 `08:00`, `12:00`, `20:00` に自動投稿が実行される。
  - **実行ラッパー**: `tedori_dotcom/run_auto_x_poster.sh`
  - **タイマー設定**: `~/Library/LaunchAgents/com.yolo-s.xposter.plist`
